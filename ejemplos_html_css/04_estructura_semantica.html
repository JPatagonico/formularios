¿Qué es el HTML Semántico?

El HTML semántico consiste en usar las etiquetas de HTML correctas para el propósito correcto. En lugar de usar <div> para todo, usamos etiquetas como <header>, <article>, y <footer>.

Beneficios: Mejora la accesibilidad (para lectores de pantalla), el SEO (para los buscadores) y la mantenibilidad del código.
